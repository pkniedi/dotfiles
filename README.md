# Setup

Configs for:

-   nvim
-   alacritty
-   i3
-   polybar
-   starship
-   zsh + oh-my-zsh
-   bin


# install
run the following command to install everything. This is not tested yet.
```sh
make install
```

# Sync all config files
Run the following command to copy the changed config files to the '.config' directory.
```sh
make sync
```
